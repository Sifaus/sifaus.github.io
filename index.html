How? What? Where?

What is this place?
This is a static website to browse Burning Wheel lifepaths, skills, traits and all that stuff. It was originally hosted on Janklabs (which is now completely empty). Before it went down, I took the code and database, completely refactored/recoded everything, and hosted on static github pages.
So what can it do?
You can check lifepaths, skills, and traits on the Explorer, you can choose lifepaths if you click the plus icon, and you can play around with your character's numbers on Burner. There are some additional options and all that in this tab, Adjuster.
Is this open source?
Yeah, you can check the code on The Source™, which can be browsed fairly easily, i think. The code lacks comments though, sorry for that.
Can I help?
Yeah, unless you have any idea how can I make this faster to load, biggest help would be testing out stuff and telling me if the character generation is actually working correctly. Also checking if all the traits, traits, and skills are included correctly would be great too. Let me know if anything is missing, open an issue on The Source™.
Can I just download the source?
Yeah, you are completely free to do whatever. You also should be able to run this offline, just download the files and run index file.
You are also adding fan-made stuff in?
Yup. I love some of those stuff. I might not always add them as they are written, but hey, you are free to take my code and do whatever with it.
Something doesn't work?
Just head over to The Source™ and open an issue.
Does this work on mobile?
I hope so. I have added some responsitivity stuff on the css but I have no clue if it actually does anything. Also the Browser (right menu, setting and list browser) should be an additional tab if you are browsing this on mobile.
What can this do?
Oh man. There are so many small functionalities this have but I don't remember all of them off the top of my head. Lets see:

Browser;
● You can choose a lifepath and add it to the Burner by clicking the plus icon if you are browsing a setting or subsetting.
● You may choose additional traits and skills and add them to your character by double clicking on them from the Skills and Traits lists. You can remove them by clicking the icon next to them that will appear on skills part of the Burner.
● Hovering over any red text will show you relevant information about it. Some red text does not have this functionality, namely the setting and list links are just bare links.

Explorer;
● You can browse settings, subsettings and lists here. Choosing what type of content you want on Adjuster will modify this menu.

Burner;
● Remove lifepaths if you added any. You can also reset and download the character as a text file.
● You can spend points and adjust skills, attributes, stats, and traits. You can also add/remove resources. A resource will be removed if you choose "none" as the type.
● Common Traits for stocks will be added automatically.
● Icons next to skills and traits will tell you what is going on with them.
● You can add your general skill points to lifepath skill points if you want to spend them that way, by clicking the plus icon next to it. Same goes for "Either Mental or Physical Pool" stat adjustment which some lifepaths have, you can choose where to spend those the same way.
● If you have the right stock and/or trait, you can choose which Emotional Attribute you want for your character. Just click the name, it should be listed there. Choosing one might add additional questions to Chronicler tab.

Chronicler;
● You can choose Appropriate Weapons and they will be added automatically. You can alter that list by moving the weapons around, the top weapon will be mandatory if the Appropriate Weapons skill is the first skill of a lifepath.
● You can answer questions, which will be reflected on Burner tab. If you choose your Emotional Attribute on Burner tab, additional questions may appear there.

Adjuster;
● You can choose settings and optional content here.

Observer;
● You are here..

There might be some additional functionality that I forgot to add here, i might add new points later.
So what is Burning Sun?
Burning Sun is a work that I compiled from the now-closed Burning Wheel forums and from various websites. It is basically a Dark Sun inspired setting for Burning Wheel. You can, probably, use it for your Dark Sun Burning Wheel games, although for me its set in a completely different world. You can download the rulebook from here.
So what is Burning Dregs?
Burning Dregs is a small compilation of lifepaths, traits, and skills. You can find the blog of the creator of Burning Dregs here.
So what is Halfling Stock?
This one is something i have found a long time ago, I have no idea who created it. You can find the original file here. If anyone knows a blog or something that I can link to, let me know.
So what is Changeling Stock?
This one is also something i have found a long time ago, and I have no idea who created it. You can find the original file here. If anyone knows a blog or something that I can link to, let me know.
Slave Raider, Black Barbarian and Lost Atlantean?
Those were official additions to the (if I am remembering correctly) Revised Edition. They were freely available and still is, I think. I adjusted the numbers on those a bit and I might have to adjust even more later (any feedback is welcome). You can download the Black Barbarian pdf from here, Lost Atlantean pdf from here, and Slave Raider from here.
